# Omix SACCO Landing Page

## Overview
Premium fintech SaaS landing page for SACCO management platform.

## Design Style
- Dark mode premium interface
- Primary accent: #FF7A1A (fintech orange)
- Secondary: #10B981 (emerald green)
- Glassmorphism cards with subtle borders

## Components
- Navigation: Fixed glass navbar
- Hero: Animated KPIs, CTA buttons
- Features: 8 feature cards
- Dashboard: Mockup preview
- Benefits: 6 key advantages
- Stats: Trust metrics
- Pricing: 3-tier with toggle
- Testimonials: SACCO manager reviews
- Security: Enterprise features
- Footer: Links and social

## Tech Stack
- React 18
- TailwindCSS
- Lucide Icons
- Inter font
